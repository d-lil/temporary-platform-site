Hexi is a mobile app that allows users to sort men's & women's clothing products by hex code.



Users can search by:


  • -Manually entering exact hex code (up to 2 colors)
  • -Selecting color from gradient wheel (up to 2 colors)
  • -Getting color palette from image upload

(with metallic/non-metallic options for all searches)
